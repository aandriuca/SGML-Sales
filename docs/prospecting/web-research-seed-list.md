# Web-Research Seed List — Canadian e-commerce prospects (all categories)

> A **starter** target list built from public web research (June 2026), across all
> e-commerce categories. Companion to the
> [`STORE-LEADS-RUNBOOK.md`](STORE-LEADS-RUNBOOK.md) (the higher-confidence source).

## ⚠️ Read this before using the list

- **Names and "Canadian" are researched; the $1M–$10M band is an ESTIMATE.** Private
  DTC brands don't publish revenue. Every "fit" below is a *hypothesis to confirm in
  Store Leads* before spending real outreach effort — not a verified figure.
- Founder names are from public sources and worth a 10-second re-check.
- This is a **seed** (~a dozen), to prove the motion and warm up `/prospect-research`
  and `/draft-outreach` — not the 500–1,000 list. That comes from Store Leads.

## What to do with this

1. Run the STRONG/POSSIBLE rows through **Store Leads** to confirm the revenue band
   and platform, and **Apollo/Clay** to get the owner's email.
2. For confirmed in-band rows, run **`/prospect-research`** for the angle, then
   **`/draft-outreach`** for the sequence.
3. Load the worked rows into SGML Sales as leads (`source = "web-research"`).
4. Treat this as the *warm-up*; the volume list (500–1,000) comes from the Store Leads
   runbook.
